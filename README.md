# 🎥 장비팀 장비관리 시스템

웹 기반의 사내 장비 대여/반납 관리 시스템입니다. 별도의 설치 없이 브라우저에서 즉시 실행 가능하며, 모바일 및 키오스크 환경에 최적화되어 있습니다.

## ✨ 주요 기능

### 🚀 대여 및 반납 (User Mode)
- **간편한 UI**: 직관적인 큰 버튼으로 빠른 대여/반납 처리
- **스마트 스캔**: RFID/바코드 스캐너 연동 지원 및 수기 입력 fallback 제공

### 🛠 관리자 기능 (Admin Mode)
- **대시보드**: 실시간 장비 입출고 현황 및 통계 시각화
- **로그 관리**: 날짜/사용자별 이력 조회 및 **Excel(CSV) 다운로드** 지원
- **자산 관리**: 보유 장비 목록 및 사용자 데이터베이스 관리

### 🎨 디자인 & 기술
- **Modern UI**: Toss 스타일의 깔끔한 디자인 시스템 적용
- **Responsive**: PC, 태블릿, 모바일 완벽 대응
- **Lightweight**: 외부 라이브러리 의존성을 최소화한 순수 HTML/JS 구조
